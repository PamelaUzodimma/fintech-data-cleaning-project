## Dataset
The dataset contains simulated financial transactions, including:
- Transaction IDs
- Account IDs
- Transaction dates
- Transaction types (debit/credit)
- Merchant IDs
- Transaction amounts
- Balances

The raw dataset is included as `original_dataset.csv`.
